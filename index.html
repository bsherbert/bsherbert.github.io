Projects

A* maze traversal

An interactive site to visually demonstrate the A* pathfinding algorithm. Allows users to create mazes or generate random ones. Animations display the path the AI is planning to take at every step of the algorithm.

Technologies Used:
HTML, CSS, Javascript, React, Node.js

Digit Classifier

A Full-Stack project built on the MERN stack. Users write answers to simple math problems in the grid. A perceptron then determines what digit was drawn by the user. The user's writings are then submitted to the database for future use by the perceptron.

Technologies Used:
HTML, CSS, Javascript, React, Node.js, Express, MongoDB, DigitalOcean
